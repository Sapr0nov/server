# server
Rapid project deployment

Grant execution rights install.sh

```bash
 chmod +x install.sh
```

Install applications and packages

```bash
 ./install.sh
``` 

Rename docker-compose.example.yml to docker-compose.yml

```bash
 cp docker-compose.example.yml to docker-compose.yml
 ```

 Run dockers

 ```bash
 docker compose up -d
```
